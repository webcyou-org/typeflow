# TypeFlow

## Usage: 

```
cargo run [OPTIONS] <MESSAGE>
```

```
typeflow [OPTIONS] <MESSAGE>
```

### Arguments

```
Vec<MESSAGE>
```

```
cargo run "Single message"
```

or 

```
cargo run "Supports" "multiple split messages"
```


### Options

```
-d, --delay <DELAY>  [default: 50]
-h, --help           Print help
```
